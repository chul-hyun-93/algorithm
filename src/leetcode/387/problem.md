387. First Unique Character in a String

문자열 s가 주어졌을 때, 첫 번째로 반복되지 않는 문자를 찾아 그 인덱스를 반환하세요. 만약 그런 문자가 존재하지 않으면 -1을 반환하세요.

예제 1:

입력: s = "leetcode"
출력: 0
예제 2:

입력: s = "loveleetcode"
출력: 2
예제 3:

입력: s = "aabb"
출력: -1


제약 조건:

1 <= s.length <= 105
s는 소문자 영어 알파벳으로만 구성됩니다.
