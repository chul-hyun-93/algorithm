주어진 크기 n의 배열 nums에서 다수 요소(majority element)를 반환하세요.

다수 요소는 배열 내에서 ⌊n / 2⌋번 이상 나타나는 요소를 의미합니다. 배열에는 항상 다수 요소가 존재한다고 가정할 수 있습니다.

예시 1:

	•	입력: nums = [3,2,3]
	•	출력: 3

예시 2:

	•	입력: nums = [2,2,1,1,1,2,2]
	•	출력: 2

제약 조건:

	•	n == nums.length
	•	1 <= n <= 5 * 10^4
	•	-10^9 <= nums[i] <= 10^9

추가 문제:

이 문제를 선형 시간과 O(1) 공간 복잡도로 해결할 수 있나요?
